# Telegram Welcome Bot

Бот приветствует новых участников группы, показывает кнопки под приветствием и позволяет настраивать текст, ссылки и свои `/команды` через админ-панель в Telegram.

## Возможности

- приветствие новых участников группы;
- подстановка имени через `{name}`;
- кнопки `Правила`, `Админы`, `Discord`, `TikTok` под приветствием;
- настройка текстов кнопок через админ-панель;
- включение и выключение отдельных кнопок;
- вход в админ-панель по паролю `zZ25255252`;
- создание своих команд, например `/info`, с текстовым ответом;
- хранение настроек в SQLite.

## Установка

1. Установите Python 3.11 или новее.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` рядом с `bot.py` по примеру `.env.example`:

```env
BOT_TOKEN=токен_от_BotFather
ADMIN_PASSWORD=zZ25255252
DB_PATH=bot.db
```

4. Запустите бота:

```bash
python bot.py
```

## Как пользоваться

1. Напишите боту в личные сообщения `/start`.
2. Отправьте пароль `zZ25255252`.
3. Настройте приветствие, тексты кнопок и команды в меню.
4. Добавьте бота в группу.
5. Выдайте боту права администратора, чтобы он видел новых участников и мог писать приветствие.

## Настраиваемые команды

В админ-панели откройте `Команды`, затем `Добавить/изменить команду`.

Пример:

```text
/discord
```

Потом отправьте текст ответа:

```text
Наша ссылка на Discord: https://discord.gg/example
```

После этого бот будет отвечать на `/discord` в личке или в группе.

## Приветствие

В тексте приветствия можно использовать `{name}`.

Пример:

```text
Привет дорогой, {name}
```
